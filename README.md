# pp_edok-bot
Бот помогает пользователям составлять меню на неделю, выбирать цели диеты (дефицит, профицит, поддержание), добавлять и управлять списком исключений продуктов, а также искать рецепты по продуктам.
# Основные функции:
Составление меню с учётом ограничений (без говядины, без рыбы, без молочки, без сахара).
Настройки диеты: добавление, удаление и просмотр продуктов-исключений.
Поиск рецептов по введённому продукту (использует Spoonacular API).
Продукты и цены (в разработке)
